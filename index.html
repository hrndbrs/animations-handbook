Animations Handbook

A reference and learning resource for building modern web animations using only HTML5, CSS3, and vanilla JavaScript (ES6+) — no frameworks, no animation libraries. Every chapter explains the theory, the browser rendering behavior behind it, a fully commented implementation, a working interactive demo, exercises, common mistakes, and performance, accessibility, mobile, and browser-compatibility notes.

Chapters

  • CSS Transitions & Transform Basics — the declarative CSS animation engine: transition, transform, and timing functions.
  • Scroll Reveal with IntersectionObserver — the observer-driven JS engine: detecting viewport entry without scroll-event polling.
  • Canvas Particle System — the imperative engine: drawing and animating directly with requestAnimationFrame and the Canvas 2D API.

CSS Properties

  • Opacity & Compositing — opacity, stacking contexts, and why it's compositor-friendly.
  • Transform Variants — translate, rotate, scale, skew, and perspective, combined live with sliders.
  • Filters & Masking — filter, backdrop-filter, clip-path, and mask.
  • Shadows & Gradients — box-shadow, text-shadow, and animated gradients via @keyframes.
  • CSS Custom Properties & @property — why animating a plain CSS variable does nothing, and the at-rule that fixes it.

Text Animations

  • Split Text Foundations — splitting text into per-character/word elements, accessibly.
  • Typing & Scramble Effects — progressive reveal and randomize-to-resolve text effects.
  • Letter/Word Reveal & Stagger — combining split text with scroll-triggered reveal.
  • Gradient Text & Underline Animation — pure-CSS decorative text effects.
  • Counter & Number Animation — eased number counting with utils/easing.js.

Mouse Interactions

  • Cursor Tracking & Mouse Follower — a lerped follower dot, and the core "smooth the pointer position" pattern.
  • Magnetic Buttons — a button pulled toward the cursor within a radius.
  • Tilt Cards — real-time 3D tilt driven by cursor position within a card.
  • Ripple & Spotlight Effects — a click ripple (keyboard-friendly) and a cursor-tracked spotlight glow.
  • Hover Reveal & Image Distortion — hover zoom and cursor-driven skew distortion.

SVG

  • Path Drawing — animating a stroke to look hand-drawn, via stroke-dashoffset.
  • Shape Morphing — turning one shape into another via matched-point interpolation.
  • Animated Icons — a checkbox, a like button, and a spinner, contrasting SMIL against CSS.
  • SVG Filters & Effects — blur, turbulence, and displacement, driven from JS since CSS variables can't reach filter internals.
  • SMIL vs CSS vs JS Animation — the same effect built three ways, to compare sequencing, control, and code volume directly.

Transform

  • Transform Deep Dive — matrix3d, transform composition order, and preserve-3d.
  • Timing & Easing — spring physics, custom cubic-bezier curves, and eased stagger sequencing.

About this project

Built phase by phase, each one adding a category and — where needed — a new shared utility:

  1. Phase 1 — site foundation, proving the documentation pattern with 3 exemplar chapters, one per animation engine.
  2. Phase 2 — CSS Properties, extending the pattern breadth-wise.
  3. Phase 3 — Text Animations, plus the site's first shared utility, utils/split-text.js.
  4. Phase 4 — Mouse Interactions, reusing utils/math.js and the pointer-tracking pattern from Phase 1's Canvas chapter across 5 pointer-driven demos.
  5. Phase 5 — SVG, the fourth core animation engine named in this project's original brief but deferred until now, across three sub-phases for depth: Path Drawing, Shape Morphing, Animated Icons, SVG Filters & Effects, and SMIL vs CSS vs JS Animation.
  6. Phase 6 — Transform Deep Dive and Timing & Easing, going deeper on the declarative CSS engine from Phase 2 and introducing a third shared utility, utils/spring.js.

Many more categories remain (more Scroll/Parallax, more Canvas effects, loading states, cards, buttons, forms, and more) and will arrive in later phases.
