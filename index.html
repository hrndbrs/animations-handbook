Animations Handbook

A reference and learning resource for building modern web animations using only HTML5, CSS3, and vanilla JavaScript (ES6+) — no frameworks, no animation libraries. Every chapter explains the theory, the browser rendering behavior behind it, a fully commented implementation, a working interactive demo, exercises, common mistakes, and performance, accessibility, mobile, and browser-compatibility notes.

Chapters

  • CSS Transitions & Transform Basics — the declarative CSS animation engine: transition, transform, and timing functions.
  • Scroll Reveal with IntersectionObserver — the observer-driven JS engine: detecting viewport entry without scroll-event polling.
  • Canvas Particle System — the imperative engine: drawing and animating directly with requestAnimationFrame and the Canvas 2D API.

About this project

This is Phase 1 of a much larger handbook. The three chapters above were chosen because they each exercise a structurally different animation engine — pure CSS, observer-driven JS, and imperative JS+Canvas — which stress-tests whether the shared documentation template and utility modules generalize before dozens more chapters are built on top of them.
